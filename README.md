## Target Audience
This repository is designed specifically for hiring managers and recruiters looking for a skilled **Data Analyst** who can leverage mathematical frameworks and IT solutions to drive operational efficiency.

## Key Highlights
* **Hybrid Skillset:** Combining analytical rigor with technical insight.
* **Core Competencies:** Python, R, MATLAB, Power BI, SQL, Pandas, Tableau.
* **Project Showcase:** Highlights real-world problem-solving including work as a Data Science Intern at SPARKS Foundation and various IBM certifications.

## Project Structure
- `intro.html`: A static introductory page featuring the newly proposed Analytical Glass UI mockup.
- `index.html`: The main entry point featuring the "About Me" and "Quick Info" sections.
- `portfolio.html`: Dynamically fetches and displays GitHub repositories.
- `resume.html`: A detailed view of education, skills, and professional experience.
- `blog.html`: Dynamic blog fetching via WordPress REST API.
- `style.css`: Contains the "Glassmorphism" styling and dark mode features.
- `script.js`: Handles dynamic content, including quotes and responsive navigation.

## Setup / Running Locally
To view the site locally without encountering CORS issues (especially for the dynamic blog fetching):

1. Clone the repository.
2. Open a terminal in the project directory.
3. Start a local server:
   ```bash
   python3 -m http.server
   ```
4. Navigate to `http://localhost:8000` in your web browser.
